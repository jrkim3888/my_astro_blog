## Tailwind CSS v4의 주요 변화

Tailwind CSS v4는 기존 버전과 비교하여 큰 변화를 가져왔습니다.
가장 큰 변화는 **CSS-first 설정** 방식입니다.

## CSS-first 설정

더 이상 `tailwind.config.js`가 필요하지 않습니다.
모든 설정을 CSS 파일에서 직접 할 수 있습니다.

```css
@import "tailwindcss";

@theme {
  --font-sans: "Pretendard", sans-serif;
  --color-primary-500: #3b82f6;
}
```

## 새로운 다크모드

`@custom-variant`를 사용하여 다크모드를 설정합니다:

```css
@custom-variant dark (&:where(.dark, .dark *));
```

이 방식은 기존의 `darkMode: 'class'` 설정을 대체합니다.

## Vite 플러그인

PostCSS 플러그인 대신 Vite 플러그인을 사용합니다:

```js
import tailwindcss from "@tailwindcss/vite";

export default {
  plugins: [tailwindcss()],
};
```

## 성능 개선

v4는 Rust로 작성된 새로운 엔진을 사용하여 빌드 속도가 크게 향상되었습니다.
특히 대규모 프로젝트에서 체감할 수 있는 성능 개선이 있습니다.

## 마이그레이션 팁

1. `tailwind.config.js` 설정을 CSS `@theme`으로 옮기기
2. PostCSS 플러그인을 Vite 플러그인으로 교체
3. `darkMode` 설정을 `@custom-variant`로 변경
4. `@apply` 사용을 최소화하고 유틸리티 클래스 직접 사용 권장

Tailwind CSS v4는 더 빠르고 간결한 개발 경험을 제공합니다.
